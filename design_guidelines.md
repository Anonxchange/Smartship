# Shipment Tracking Website Design Guidelines

## Design Approach
**Reference-Based Approach** - Drawing inspiration from professional logistics platforms like FedEx, DHL, and UPS, emphasizing trust, reliability, and corporate professionalism. The design should convey global reach and operational excellence.

## Core Design Elements

### A. Color Palette
**Primary Colors:**
- Deep Navy Blue: 220 85% 15% (headers, navigation, primary buttons)
- Professional Blue: 220 75% 35% (accents, links, secondary elements)
- Light Blue: 220 60% 92% (backgrounds, subtle highlights)

**Supporting Colors:**
- Pure White: 0 0% 100% (main backgrounds, text areas)
- Charcoal Gray: 220 15% 25% (body text, secondary information)
- Light Gray: 220 10% 85% (borders, dividers, input backgrounds)
- Success Green: 120 65% 45% (delivery status, positive indicators)
- Warning Orange: 35 85% 55% (transit status, alerts)

### B. Typography
- **Primary Font:** Inter or similar modern sans-serif from Google Fonts
- **Headings:** Bold weights (600-700) for trust and authority
- **Body Text:** Regular weight (400) with excellent readability
- **Labels/UI:** Medium weight (500) for form elements and navigation

### C. Layout System
**Tailwind Spacing:** Use units of 4, 6, 8, 12, and 16 for consistent spacing
- Headers and sections: py-12 to py-16
- Content spacing: gap-8 between major elements
- Cards and components: p-6 to p-8
- Form elements: p-4 for inputs and buttons

### D. Component Library

**Navigation:**
- Clean horizontal navigation with company logo
- Sticky header with track shipment quick access
- Professional dropdown menus for services

**Hero Section:**
- Large banner with shipping/logistics imagery
- Prominent tracking number input field
- Clear value propositions with trust indicators

**Service Cards:**
- Grid layout showcasing different shipping methods
- Icons representing each service type
- Clean borders and subtle shadows

**Statistics Dashboard:**
- Large numbers with descriptive labels
- Counter animations for delivered packages
- Professional metric presentation

**Forms:**
- Clean input fields with proper spacing
- Primary blue submit buttons
- Form validation with helpful messaging

**Testimonials:**
- Customer photos and quotes
- Company attribution with roles
- Star ratings or satisfaction indicators

### E. Images
**Required Images:**
- **Hero Image:** Large banner showing shipping containers, cargo planes, or global logistics operations (full-width hero)
- **Service Icons:** Professional icons for air freight, sea freight, road transport, warehousing
- **Statistics Background:** Subtle logistics-themed background patterns
- **Customer Photos:** Professional headshots for testimonials
- **Partner Logos:** Industry partner/certification logos
- **Global Map:** World map showing coverage areas

**Image Treatment:**
- Professional photography with blue color overlays
- Consistent aspect ratios across service cards
- High-contrast images that support text overlay
- Corporate/industrial aesthetic throughout

## Key Design Principles

1. **Trust & Reliability:** Use consistent spacing, professional imagery, and established UI patterns
2. **Global Scale:** Emphasize worldwide coverage through maps, statistics, and international elements  
3. **Operational Excellence:** Clean, efficient layouts that reflect logistics precision
4. **Easy Tracking:** Make shipment tracking the primary call-to-action throughout the site
5. **Corporate Professionalism:** Maintain business-appropriate tone with authoritative typography and structured layouts

## Critical Requirements
- Responsive design optimizing for mobile tracking usage
- Clear visual hierarchy emphasizing tracking functionality
- Professional color scheme building trust and authority
- Comprehensive service showcase with equal visual weight
- Statistics and achievements prominently displayed
- Customer testimonials with authentic presentation
